# SaaS App Hero Section

A responsive hero section built using **HTML5** and **Tailwind CSS**.  
The project recreates a modern SaaS/app landing page layout with a headline, description, call-to-action buttons, and an app mockup image.

## Tech Used
- HTML5  
- Tailwind CSS (CDN)

## Features
- Clean and modern design  
- Fully responsive (mobile & desktop)  
- No JavaScript
